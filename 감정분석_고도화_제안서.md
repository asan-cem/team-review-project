
#### **의료 협업 맥락 특화 감정**

- **신뢰**: "믿고 맡길 수 있어요", "전문적이에요"
- **만족**: "만족스러워요", "좋았습니다"
- **분노**: "화가 나요", "짜증나요"

## 🛠️ 기술 구현 방안

### **방법 1: 프롬프트 엔지니어링 고도화**
```python
ADVANCED_PROMPT = """
다음 협업 후기를 8가지 세분화된 감정으로 분석하세요:

[기본 분석]
- primary_emotion: 주요 감정 (기쁨/감사/신뢰/만족/분노/슬픔/두려움/실망)
- emotion_intensity: 감정 강도 (1-10)
- emotional_complexity: 단순/복합

[고급 분석]
- secondary_emotion: 보조 감정 (복합 감정인 경우)
- emotion_context: 의료협업 맥락 (환자안전/업무효율/인간관계/전문성)
- root_cause: 감정 원인 (부정 감정인 경우)
- improvement_suggestion: 개선 방안 제안
"""
```

```python
}
```

### **방법 3: 규칙 기반 + AI 하이브리드**
```python
def 고도화_감정분석(텍스트):
    # 1단계: 기존 AI 분석

    # 4단계: 통합 결과 생성
```
