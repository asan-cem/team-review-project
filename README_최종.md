# 🏥 서울아산병원 협업평가 분석 시스템

## 🎯 **프로젝트 개요**
서울아산병원의 협업평가 설문조사 데이터를 AI로 분석하고 인터랙티브 대시보드를 생성하는 시스템입니다.

---

## 🚀 **빠른 시작 (5분 가이드)**

### **1️⃣ 데이터 분석 (10-30분 소요)**
```bash
python main.py
```

### **2️⃣ 대시보드 생성 (10초 소요)**
```bash
python build_dashboard_html_improved.py
```

### **3️⃣ 결과 확인**
- `서울아산병원 협업평가 대시보드.html` 파일을 웹 브라우저에서 열기

## ⚡ **주요 기능**

### **🤖 AI 데이터 분석 (main.py)**
- 감정 분석 (긍정/부정/중립)
- 핵심 키워드 추출
- 감정 강도 점수 계산
- 의료 맥락 분석

### **📊 인터랙티브 대시보드**
- 연도별/부문별/부서별 분석
- 실시간 필터링
- 키워드 클릭시 관련 리뷰 표시
- 반응형 웹 디자인
- 보안 강화 (XSS 방지)

## 🔄 **일반적인 작업 흐름**

```mermaid
graph LR
    A[새 엑셀 데이터] --> B[main.py 실행]
    B --> C[AI 분석 완료]
    C --> D[대시보드 생성]
    D --> E[웹 대시보드 완성]
    E --> F[공유/프레젠테이션]
```

1. **데이터 준비**: 새 설문조사 엑셀 파일을 프로젝트 폴더에 복사
2. **AI 분석**: `main.py` 실행하여 감정분석 및 키워드 추출
3. **대시보드 생성**: `build_dashboard_html_improved.py` 실행
4. **결과 활용**: HTML 파일을 웹 브라우저에서 열어 분석 결과 확인
