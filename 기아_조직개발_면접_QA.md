### Q1: "조직문화진단 도구를 자체 개발하셨다고 했는데, 외부 용역 대비 어떤 점이 개선되었나요?"

**답변:**

**배경:**
- 2019년 KMAC(한국능률협회컨설팅) 용역으로 진단 도구 개발 (2022년까지 사용)
- 문제점: 의료기관 특수성(의사직, 간호직, 사무직 등 다양한 직군) 반영 부족

**자체 개발 과정:**
```
1. 조직문화진단 논문 리서치
   - Denison, Kliman & Saxton, Cameron & Quinn, Ginevicius & Vaitkunaite 등

2. 문항 설계
   - 100여 개 문항 pool 구성
   - 최종 45문항으로 정제

3. 통계적 검증
   - 타당도 조사: 경영진, 원내 교수 대상
   - 신뢰도 검증

4. 시행 및 분석
   - 표본 수: 약 3,000건
   - 조직문화 개선 계획 수립
```

**개선 효과:**
1. **적합성:** 의료기관 특성 반영 → 의사직/간호직/사무직 모두 의미 있는 응답
2. **활용도:** 진단 결과가 실제 개선과제로 직접 연결
   - 심리적 안전감 프로젝트 런칭
   - 역지사지 프로그램 신설
3. **비용:** 외부 용역 연간 ₩30M → 자체 운영 ₩0 (인건비 제외)
4. **속도:** 문항 수정 요청 시 즉시 반영 가능 (용역은 3개월 소요)

**기아 적용 방안:**
- 기아의 제조업 특성 반영한 맞춤형 진단도구 개발
- R&D, 생산, 영업, 지원 부문 간 차이 고려
- 글로벌 법인 고려 시 문화적 차이 반영

### Q2: "부서간 협업평가에서 연간 2만여 건의 데이터를 분석한다고 했는데, 어떻게 인사이트를 도출하나요?"

**답변:**

**데이터 규모:**
- 연간 2회 시행 (상반기 9,000건 + 하반기 9,000건 = 약 18,000건)
- 2022-2025년 누적 약 46,654건
- 5개 평가 문항 + 주관식 의견

**분석 프로세스:**

**1단계: 정량 데이터 분석**
```
평가 문항:
1. 존중배려: 상대 부서가 우리 부서를 존중하는가?
2. 정보공유: 필요한 정보를 적시에 공유하는가?
3. 명확처리: 업무 요청을 명확하게 처리하는가?
4. 태도개선: 개선이 필요한 태도가 있는가?
5. 전반만족: 전반적인 협업 만족도

분석 방법:
- 부서별 평균점수: 연도별/반기별 추이 분석
- 상호평가 쌍 분석: A→B vs B→A 점수 비교
- 이상치 탐지: 표준편차 2배 이상 차이 부서 파악
```

**2단계: 정성 데이터 분석 (AI 활용)**
```python
# LLM 기반 주관식 의견 자동 분류
감정 분류: 긍정(3,421건) / 부정(1,892건) / 중립(3,687건)
핵심 키워드: 협업 우수사례, 프로세스 개선 필요사항
의료 맥락: 환자 안전, 진료 품질, 업무 효율

예시:
원문: "응급실에서 급한 검사 요청 시 항상 빠르게 대응해주셔서 환자 치료에 큰 도움이 됩니다."
→ 감정: 긍정, 키워드: ['빠른 대응', '환자 치료'], 맥락: 환자 안전
```

**3단계: 인사이트 도출**
```
1. 부서별 강점/약점 파악
   - 간호부 ↔ 검사의학과: 존중배려 4.8점 (우수)
   - 행정부 ↔ 진료부: 정보공유 3.2점 (개선 필요)

2. 협업 패턴 발견
   - 물리적 거리가 가까운 부서: 평균 4.5점
   - 디지털 소통만 하는 부서: 평균 3.8점
   → 역지사지 프로그램 현장 방문의 중요성 입증

3. 시계열 변화 추적
   - 역지사지 프로그램 시행 전후 비교
   - 2024 상반기: 3.2점 → 2024 하반기: 4.1점 (+28%)
```

**실시간 모니터링:**
- React 대시보드로 부서장이 실시간 확인
- 평가 시행 중 응답률 모니터링
- 이슈 발생 시 즉시 경영진 알림

**기아 적용 방안:**
- 생산부문 ↔ 품질부문 협업 평가
- 글로벌 법인 간 협업 품질 측정
- 데이터 기반 조직개발 플랫폼 구축

### Q3: "16개 그룹 70명의 심층 인터뷰를 진행했다고 했는데, 어떻게 설계하고 분석했나요?"

**답변:**

**배경 및 목적:**
- 조직문화 개선 간담회 (2023년)
- 목적: 협업, 존중, 직원 중심적 근무환경에 대한 세대/직역별 인식 파악

**인터뷰 설계:**

**1. 참가자 모집 전략**
```
직역별 × 연령대별 매트릭스:

             20대   30대   40대   50대
간호직       [그룹1] [그룹2] [그룹3] [그룹4]
보건직       [그룹5] [그룹6] [그룹7] [그룹8]
사무직       [그룹9] [그룹10] [그룹11] [그룹12]
기능/기술직  [그룹13] [그룹14] [그룹15] -
의사직       - [그룹16] - -

총 16그룹 × 4-5명 = 70명

모집 방법:
- 홍보 포스터 제작 (미리캔버스)
- 사내 게시판 공지
- 부서장 추천 + 자발적 신청
```

**2. 인터뷰 질문 구조**
```
파트 1: 조직문화 인식 수준 (15분)
- "우리 병원의 조직문화를 한 단어로 표현한다면?"
- "협업, 존중, 심리적 안전감에 대해 어떻게 느끼시나요?"

파트 2: 인식 저해요인 파악 (20분)
- "조직문화 개선이 어려운 구조적 원인은?"
- "세대 간 / 직역 간 갈등 사례는?"

파트 3: 개선 의견 및 발전 방안 (15분)
- "가장 시급하게 개선해야 할 것은?"
- "실행 가능한 구체적 방안 제안"

파트 4: 기타 의견 (10분)
- "경영진에게 전하고 싶은 말씀"
```

**3. 분석 프로세스**
```
정성 분석:
1. 녹취록 작성 (16그룹 × 1시간 = 약 240페이지)
2. 주제별 코딩 (Thematic Analysis)
   - 협업: 부서간 소통 부족, 정보 비대칭
   - 존중: 호칭 문제, 세대 차이
   - 심리적 안전감: 수직적 문화, 실수 두려움
3. 직역별 / 연령별 차이 분석
4. 빈도 분석: 가장 많이 언급된 키워드

정량 분석:
- 각 주제별 언급 빈도
- 긍정/부정 의견 비율
- 세대별 인식 차이 통계
```

**4. 경영진 보고**
```
보고서 구성:
1. 주요 발견사항 (Executive Summary)
2. 직원 인식도 (주제별 긍정/부정 비율)
3. 저해 요인 (구조적 원인)
4. 개선 의견 (실행 가능 과제)
5. 기타 의견 (경영진 참고사항)

보고 방식:
- 파워포인트 40페이지
- 핵심 인용구 활용
- 데이터 시각화 (워드클라우드, 막대그래프)
```

**5. 실행 과제 도출**
```
2024년 조직문화 개선 과제 반영:

인터뷰 인사이트 → 실행 과제
"부서간 업무 이해 부족" → 역지사지 프로그램 신설
"리더의 실수 수용 부족" → 심리적 안전감 리더십 교육
"호칭 혼란" → 호칭 지침서 배포
"세대 차이 갈등" → 세대 존중 특강 시행
```

**성과:**
- 직원 참여도: 신청자 120명 중 70명 선발 (1.7:1 경쟁률)
- 실행률: 인터뷰에서 도출된 개선의견 12건 중 8건 실행 (67%)

**기아 적용 방안:**
- 생산직 ↔ 사무직 간 문화 차이 인식 조사

### Q4: "연간 1,000건 이상의 사내 블라인드를 관리한다고 했는데, 구체적으로 어떻게 대응하나요?"

**답변:**

**사내 블라인드 현황:**
- 병원 내 임직원 전용 익명 게시판
- 연간 약 1,000건 이상 게시글
- 주요 주제: 복지, 업무 프로세스, 조직문화, 불만/건의

**대응 프로세스:**

**1단계: 모니터링 및 분류 (매일)**
```
게시글 분류 체계:

긴급도:
- 🚨 긴급: 환자 안전, 법적 리스크 (1시간 내 보고)
- ⚠️ 중요: 업무 효율, 직원 불만 (당일 검토)
- ℹ️ 일반: 건의사항, 질문 (주간 취합)

카테고리:
- 복지 제도: 휴가, 급여, 복리후생
- 업무 프로세스: 시스템, 절차 개선
- 조직문화: 협업, 소통, 존중
- 시설/환경: 주차, 식당, 편의시설
- 기타: 경영진 소통, 정책 건의
```

**2단계: 관련 부서 협의**
```
예시: 병원 발신번호 표시 서비스 건의

블라인드 게시글 (2023.03):
"병동에서 환자 보호자가 대표번호 대신 병동으로 직접 전화해요.
업무에 집중할 수 없고, 환자 케어가 어렵습니다.
발신번호 표시 서비스 도입해주세요."

대응 과정:
1. 유사 게시글 검색: 과거 6개월간 23건 발견
2. 현황 파악:
   - 간호부 7개 병동 설문: 월평균 150건 불필요한 전화
   - 업무 가중 시간: 간호사 1인당 월 2시간
3. 경영진 보고:
   - 문제점: 환자 직접 전화 → 간호 업무 방해
   - 비용: 연간 ₩5M (통신비)
   - 효과: 간호사 업무 효율 개선, 환자 만족도 향상
4. 결과: 사업 재개 승인 → 3개월 내 도입

블라인드 피드백:
"발신번호 서비스 도입되었습니다! 건의 들어주셔서 감사합니다 👍"
→ 좋아요 342개
```

**3단계: 경영진 보고 및 조치**
```
주간 보고:
- 이슈 요약: 주요 5건
- 트렌드 분석: 이번 주 핫 이슈
- 조치 현황: 해결 완료 / 진행 중

월간 보고:
- 통계 분석: 카테고리별 게시글 수
- 감정 분석: 긍정/부정 비율
- 개선 과제: 반복되는 불만사항

경영진 회의 (조직문화위원회):
- 구조적 문제: 제도 개선 필요 사항
- 예산 필요 사항: 투자 우선순위 검토
```

**4단계: 피드백 및 공유**
```
블라인드 답변:
- 익명 게시 → 공식 답변 어려움
- 대신 사내 게시판에 공지
- "○○○ 건의사항 검토 결과 안내"

정책 반영 사례:
1. 주차 공간 부족 → 외부 주차장 임차 (2023)
2. 직원 식당 메뉴 개선 → 메뉴 투표제 도입 (2024)
3. 자녀 교육 상담 → 복지 제도 신설 (2022)
```

**성과 지표:**
```
대응률: 95% (1,000건 중 950건 검토 완료)
해결률: 62% (긴급/중요 건 중 62% 조치 완료)
만족도: 블라인드 긍정 댓글 비율 78% (2024년 기준)
```

**기아 적용 방안:**
- 기아 사내 소통 플랫폼 운영 경험 활용
- 데이터 기반 이슈 트렌드 분석
- 경영진-직원 간 소통 중재 역할

### Q5: "역지사지 프로그램을 신설했다고 했는데, 어떻게 기획하고 실행했나요?"

**답변:**

**배경:**
- 조직문화 개선 간담회(2023)에서 "부서간 업무 이해 부족" 이슈 도출
- 블라인드 게시판에서 "타부서가 우리 일을 몰라서 무리한 요구" 빈번

**프로그램 목적:**
```
환자를 중심으로 제공되는 서비스 향상을 위해
유관부서 간의 현장 방문 및 소통을 중재하여
타부서 업무 환경과 입장을 이해시키고,
협업을 원활하게 하기 위함
```

**개발 프로세스:**

**1단계: 직원 의견 조사 (2024.06)**
```
설문 대상: 부서간 협업 빈도 높은 부서 (30개 부서, 450명)

질문:
1. 타부서 업무 이해도: 평균 2.8/5.0 (낮음)
```

```python
# A → B vs B → A 점수 비교
def find_mutual_pairs(df):
    results = []

    for _, row in df.iterrows():
        team_a = row['평가_부서명']
        team_b = row['피평가대상 부서명']
        score_a_to_b = row['종합점수']

        # B → A 평가 찾기
        mutual = df[(df['평가_부서명'] == team_b) &
                    (df['피평가대상 부서명'] == team_a)]

        if not mutual.empty:
            score_b_to_a = mutual['종합점수'].values[0]
            diff = abs(score_a_to_b - score_b_to_a)

            results.append({
                '부서 A': team_a,
                '부서 B': team_b,
                'A→B 점수': score_a_to_b,
                'B→A 점수': score_b_to_a,
                '점수 차이': diff
            })

    return pd.DataFrame(results)

mutual_pairs = find_mutual_pairs(df)
# 점수 차이 큰 순으로 정렬
mutual_pairs.sort_values('점수 차이', ascending=False, inplace=True)

print(mutual_pairs.head(10))

# 출력:
```
