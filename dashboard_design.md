# 협업 평가 대시보드 고도화 제안 (참고용.md 반영)

## 1. 대시보드 주요 기능 및 구조

### 1) 필터링 기준 (기준 컬럼)
- **설문시행연도**: 연도별 분석 및 비교 (다중 선택 가능)
- **평가_부문/부서/Unit**: 평가를 실시한 조직 단위별 분석 (다중 선택 가능)
- **피평가대상 부문/부서/Unit**: 평가를 받은 조직 단위별 분석 (다중 선택 가능)
- **문항별 필터링**: 특정 객관식 문항 선택하여 분석 (예: '존중배려', '정보공유' 등)
- **위치**: 대시보드 상단 또는 사이드바에 위치하여 다중 선택/필터링 가능하도록 구현

### 2) 객관식 점수 분석
- **연도별 문항 점수 분석**:
    - 전체 및 부문별 연도별 점수 추이 (라인 차트, 표)
    - 각 문항별 평균, 분포 (히스토그램 또는 박스 플롯)
- **부문 비교 분석**:
    - 특정 연도의 모든 부문 점수 비교 (막대 그래프, 표)
    - 부문 간 상대적 성과 분석
- **상세 분석 (부서/Unit별)**:
    - 연도, 부서, Unit을 선택하여 세부 분석
    - 주요 지표 (평균 점수, 응답 수, 표준편차) 확인
    - **상호평가 매트릭스**: 평가↔피평가 부서/Unit 간 점수 및 관계 시각화 (히트맵 또는 테이블)
- **시각화**: 표, 막대그래프, 라인차트, 히트맵 등 활용

### 3) 주관식 분석 컬럼 활용 (정성적 인사이트 강화)
- **정제된_텍스트**: 주관식 응답의 전처리/정제 결과 (대표 의견 예시 리스트 - 긍정/부정/중립별)
- **감정_분류**: 긍정/부정/중립 등 감정 카테고리 (파이 차트, 바 차트, 연도별 변화 라인 차트)
- **감정_강도_점수**: 감정의 강도 (히스토그램 또는 밀도 플롯, 조직별/연도별 평균 비교)
- **핵심_키워드**: 주요 키워드 추출 (워드클라우드, 키워드별 빈도표)
- **의료_맥락**: 의료 현장 특화 맥락 분류 (의료 특화 카테고리별 응답수/감정 분포 바 차트)
- **시각화**: 주관식 응답의 정성적 인사이트, 조직별/연도별 감정 경향, 키워드 트렌드 등 시각화

## 2. 대시보드 UI/UX 설계 예시 (완벽한 UI를 위한 구성)

- **레이아웃**:
    - 상단: 필터 영역 (연도, 평가/피평가 부문/부서/Unit, 문항별 다중 선택 드롭다운 또는 체크박스)
    - 메인 콘텐츠 영역: 객관식 점수 분석, 주관식 분석 결과 섹션으로 명확히 분할
- **객관식 섹션**:
    - **종합 현황**: 전체 및 부문별 연도별 점수 추이 (라인 차트, 표)
    - **문항별 상세**: 각 문항별 평균, 분포, 상위/하위 조직 (막대 그래프, 히스토그램, 정렬 가능한 표)
    - **부문 비교**: 특정 연도 부문 간 점수 비교 (막대 그래프)
    - **상세 분석**: 선택된 부서/Unit의 주요 지표 및 상호평가 매트릭스 (표, 히트맵)
- **주관식 섹션**:
    - **감정 분석 요약**: 감정 분류 파이/바 차트, 연도별 감정 변화 라인 차트
    - **핵심 키워드**: 워드클라우드, 키워드별 빈도표
    - **의료 맥락 분석**: 의료 특화 카테고리별 응답수/감정 분포 바 차트
    - **대표 의견**: 긍정/부정/중립별 실제 응답 예시 리스트 (스크롤 가능)
- **반응형 디자인**: 다양한 화면 크기(데스크톱, 태블릿)에 최적화된 레이아웃 제공 및 대화형 요소 강조

## 3. 주요 시각화/기능 예시

- **필터**: 연도, 평가/피평가 부문·부서·Unit, 문항 다중선택
- **객관식 점수**:
    - 종합점수/문항별 점수: 표, 막대그래프, 라인차트, 히트맵
    - 상호평가 매트릭스(평가↔피평가): 히트맵 또는 테이블
- **주관식 분석**:
    - 감정 분류/강도: 파이차트, 바차트, 연도별 변화 라인 차트
    - 핵심 키워드: 워드클라우드, 키워드별 빈도표
    - 의료 맥락별 분포: 의료 특화 카테고리별 응답수/감정 분포 바 차트
    - 대표 의견: 긍정/부정/중립별 예시 리스트
- **요약/보고서 (선택 사항)**: 주요 인사이트 자동 요약, PDF/Word 보고서 연동 기능

## 4. 기술 스택 추천

- **프레임워크**: Python + Streamlit (대화형 필터, 표/그래프/워드클라우드, 주관식 분석 결과 시각화에 용이)
- **데이터 처리**: Pandas/Numpy (데이터 집계, 전처리, 감정/키워드 분석)
- **시각화**: Plotly (대화형 차트 생성에 최적화), Altair/Seaborn (보조 시각화)
- **텍스트 분석**: NLTK, KoNLPy, transformers 등 (주관식 텍스트 감정/키워드 분석 - 사전 분석 결과 활용)
- **배포/공유 (선택)**: Power BI, Tableau (사내 배포/공유 목적 고려 시)

## 5. 결론

본 대시보드는 "기준(필터링) + 객관식 점수 + 주관식 분석"의 3대 축을 모두 반영하며, `참고용.md`에서 제시된 성공적인 UI/UX 요소를 통합하여 2025년 협업평가 결과 보고서 작성 및 조직 인사이트 도출에 최적화될 것입니다. 실시간 필터링, 다양한 시각화(표, 그래프, 워드클라우드), 감정/키워드/의료 맥락별 분석을 모두 한 화면에서 확인할 수 있도록 설계하여 가장 효과적인 사용자 경험을 제공할 것입니다.
