## 🔍 수정 항목별 가이드라인

### **1. 정제텍스트 수정**

#### ❌ 수정이 필요한 경우:
- **오타 및 맞춤법**: "감사드림니다" → "감사드립니다"
- **불완전한 문장**: "업무가..." → "업무 처리가 신속하지 않음"
- **비속어/부적절한 표현**: "짜증남" → "개선이 필요함"
- **의미 불명확**: "그냥..." → "특별한 의견 없음"

#### ✅ 수정 원칙:
- 원본 의미 보존
- 전문적이고 정중한 표현으로 변경
- 구체적이고 명확한 표현 사용

### **2. 감정분석 수정**

#### 감정 분류 기준:
```
긍정: 만족, 감사, 칭찬, 협력적 표현
- 예: "친절하게 도와주셔서 감사합니다"
- 예: "항상 신속하게 처리해주십니다"

중립: 사실 기술, 제안, 건설적 피드백
- 예: "업무 처리 시간이 다소 소요됨"
- 예: "정보 공유가 더 원활했으면 좋겠습니다"

부정: 불만, 비판, 개선 요청
- 예: "의사소통이 원활하지 않음"
- 예: "업무 처리가 지연되는 경우가 많음"
```

### **3. 감정강도 수정 (1-10 척도)**

#### 강도 기준:
```
1-3점 (약함): 미미한 감정 표현
- "조금 아쉬움", "약간 만족"

4-6점 (보통): 일반적인 감정 표현
- "만족합니다", "개선이 필요합니다"

7-10점 (강함): 강한 감정 표현
- "매우 만족", "심각한 문제", "대단히 감사"
```

### **4. 분류라벨 수정**

#### 5가지 분류 기준:
```
1. 부서간 협업: 부서 간 업무 조율, 협력 관련
2. 직원간 소통: 개인 간 의사소통, 정보 전달
3. 전문성 부족: 업무 역량, 기술적 문제
4. 업무 태도: 근무 자세, 서비스 마인드
5. 상호 존중: 예의, 배려, 인간관계
```

### **5. 품질점수 수정 (1-10 척도)**

#### 품질 평가 기준:
```
1-3점: 분석 결과 신뢰도 낮음
- 모호한 표현, 맥락 파악 어려움

4-6점: 보통 수준의 신뢰도
- 일반적인 분석, 약간의 불확실성

7-10점: 높은 신뢰도
- 명확한 의미, 정확한 분석 결과
```

## 📝 수정 작업 절차

### Step 1: 검토 항목 식별
```excel
1. Excel에서 '재검토필요' = True 필터 적용
2. 'quality_issues' 컬럼에서 문제점 확인
3. 우선순위 설정 (품질점수 낮은 순)
```

### Step 2: 원본 텍스트 분석
```
1. '협업 후기' 컬럼의 원본 텍스트 읽기
2. 맥락과 의도 파악
3. 부서 관계 및 상황 고려
```

### Step 3: 체계적 수정
```excel
1. 정제텍스트 수정
2. 감정분석 재분류
3. 감정강도 재평가
4. 분류라벨 재할당
5. 품질점수 업데이트 (7점 이상 목표)
6. 재검토필요 = False로 변경
```

### Step 4: 수정 근거 기록
```excel
추가 컬럼 생성: '수정사유'
- 예: "감정분석 오류 수정: 부정→중립"
- 예: "오타 수정 및 표현 개선"
- 예: "분류라벨 변경: 소통→태도"
```
