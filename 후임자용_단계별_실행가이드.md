## 🚀 **STEP 1: 새로운 설문조사 데이터 받기**

### **📥 데이터 파일 준비**

1. **새 설문조사 엑셀 파일을 받으셨나요?**
   - 파일명 예시: `설문조사_전처리데이터_20250115_1200.xlsx`
   - 파일 크기: 보통 5-50MB

2. **파일을 올바른 위치에 복사하기**
   ```
   📂 team-review-project/
   └── 📄 설문조사_전처리데이터_20250115_1200.xlsx  ← 여기에 복사!
   ```

3. **파일명 확인하기**
   - ✅ 올바른 예: `설문조사_전처리데이터_20250115_1200.xlsx`
   - ❌ 잘못된 예: `새 파일.xlsx`, `data.xlsx`

### **🔧 파일명이 다른 경우 처리법**

만약 받은 파일명이 다르다면:

**방법 1: 파일명 변경 (권장)**
1. 파일을 우클릭 → "이름 바꾸기"
2. `설문조사_전처리데이터_YYYYMMDD_HHMM.xlsx` 형식으로 변경
   - YYYY: 연도 (예: 2025)
   - MM: 월 (예: 01)
   - DD: 일 (예: 15)
   - HH: 시간 (예: 12)
   - MM: 분 (예: 00)

**방법 2: 설정 파일 수정**
1. `dashboard_config.py` 파일을 메모장으로 열기
2. `DATA_FILE = "기존파일명.xlsx"` 찾기
3. `DATA_FILE = "새파일명.xlsx"`로 변경
4. 저장하기

## 📤 **STEP 5: 결과 공유하기**

### **🎯 공유 방법 선택**

#### **1. HTML 파일 직접 공유**
```
장점: 상호작용 기능 모두 사용 가능
방법: 
1. HTML 파일을 이메일 첨부
2. 받는 사람이 웹 브라우저에서 열기

```

#### **2. 스크린샷으로 공유**
```
장점: 간단하고 용량이 작음
방법:
1. 웹 브라우저에서 대시보드 열기
2. 스크린샷 캡처 (Win: Win+Shift+S, Mac: Cmd+Shift+4)
3. 이미지 파일로 공유
```

#### **3. PDF로 변환하여 공유**
```
장점: 인쇄 가능, 공식 문서 형태
방법:
1. 웹 브라우저에서 Ctrl+P (Mac: Cmd+P)
2. "PDF로 저장" 선택
3. PDF 파일 생성 후 공유
```

### **📧 이메일 첨부시 주의사항**

1. **파일 크기 확인**
   - HTML 파일은 보통 50KB 이하
   - 이메일 첨부 제한(보통 25MB) 이내인지 확인

2. **받는 사람 안내**
   ```
   "첨부된 HTML 파일을 다운로드 후 더블클릭하여 웹 브라우저에서 열어주세요.
   모든 차트와 필터 기능을 사용할 수 있습니다."
   ```

## 🚨 **문제 발생시 대처법**

### **❌ 자주 발생하는 문제와 해결책**

#### **1. "파일을 찾을 수 없습니다"**
```
🔍 증상: FileNotFoundError 메시지
💡 해결:
1. 파일명이 정확한지 확인
2. 파일이 올바른 폴더에 있는지 확인
3. 파일명에 한글이나 특수문자가 있는지 확인
```

#### **2. "프로그램이 너무 오래 걸려요"**
```
🔍 증상: 30분 이상 실행 중
💡 해결:
1. 정상적인 현상일 수 있음 (데이터 양에 따라)
2. 진행률 표시 확인
3. 다른 프로그램들 종료해서 메모리 확보
```

#### **3. "대시보드가 비어있어요"**
```
🔍 증상: 웹 페이지는 열리지만 차트가 없음
💡 해결:
1. 페이지를 새로고침 (F5)
2. 다른 웹 브라우저에서 시도
3. 인터넷 연결 확인
4. JavaScript 활성화 확인
```

#### **4. "권한이 없다고 나와요"**
```
🔍 증상: PermissionError 메시지
💡 해결:
1. 엑셀 파일이 열려있다면 닫기
2. 관리자 권한으로 실행
3. 폴더 권한 확인
```
